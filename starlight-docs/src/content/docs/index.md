Welcome to OpenSearch AgentOps! This guide will help you get up and running quickly.

## What is OpenSearch AgentOps?

OpenSearch AgentOps is an OpenTelemetry-based observability platform for AI applications. It provides:

- **Distributed Tracing**: Track requests across your AI system
- **Evaluation**: Run experiments and compare model performance
- **Prompt Management**: Version and deploy prompts with confidence
- **Agent Observability**: Visualize agent reasoning and tool calls
